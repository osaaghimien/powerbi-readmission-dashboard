# Project Title  
Healthcare Readmission Analysis Dashboard  

## Overview  
This project analyzes hospital readmission data to identify patterns, track key performance metrics, and support better care and cost decisions through an interactive dashboard.  

## Business Problem  
High 30-day readmission rates increase hospital costs and indicate gaps in patient care. Hospitals need a way to identify high-risk patients and reduce avoidable readmissions.  

## Dataset  
Healthcare dataset containing patient admissions, discharge details, readmission status, and length of stay.  

## Tools Used  
Power BI, SQL Server, DAX, Excel  
## Key Analysis  
Analyzed readmission trends, high-risk patient segments, and length of stay to identify patterns driving repeat hospital visits.  

## Results and Insights  
Found that a small group of high-risk patients contributed to a large portion of readmissions, highlighting the need for targeted follow-up care.  

## Deliverables  
Interactive Power BI dashboard with KPI tracking, trend analysis, and risk segmentation insights.  

## What I Learned  
Strengthened my ability to translate business problems into measurable KPIs and communicate insights clearly through dashboards.
